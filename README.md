# INSTALACJA:
## Dla chromium:
1) Pobierasz najnowszy `.zip` [release](https://github.com/GieniuG/SimpleExamBrowserSpoofer/releases) (lub to repo)
3) Rozpakujesz plik zip
2) Wchodzisz w ustawienia wtyczek
3) Włączasz developer mode 
4) Klikasz "Load unpacked" 
5) Wybierasz folder z tym rozszerzeniem i się cieszysz 
## Dla firefox:
1) Pobierasz najnowszy `.xpi` [release](https://github.com/GieniuG/SimpleExamBrowserSpoofer/releases)
2) Wchodzisz w ustawienia wtyczek
3) Install Add-on From File
4) Wybierasz plik .xpi
5) Klikasz "Add"
